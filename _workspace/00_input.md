# 모두의 솔로몬 — 빌드 인풋

## 프로젝트 개요
유저들이 일상적인 갈등/딜레마를 올리고 24시간 동안 투표를 받아 결과를 확인하는 가벼운 엔터테인먼트 투표 커뮤니티.

## 핵심 요구사항
- 소셜 로그인(Google) + 이메일 회원가입/로그인
- 게시글 작성: 내용 + 이미지(선택) + 2~4개 선택지, expires_at = now + 24h
- 24시간 블라인드 룰: expires_at 전까지 득표 수 절대 비공개
- 투표: 1인 1게시글 1회 제한
- 결과 공개: expires_at 후 솔로몬의 선택(최다 득표) 하이라이트
- 솔로몬 지수: majority_matched_votes / total_votes × 100
- 칭호 시스템: 지수 구간별 칭호 부여
- PWA (모바일 설치 가능)
- Vercel 배포
